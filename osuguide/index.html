What is Osu?

Osu is a rhythm game that is avalible on Windows and Mac(Linux requries something else). It has four different gamemode for the players to choose from such as, Osu standard which aim game which focuses around clicking circles. CTB or Catch the beat, which is a game where you control a type of tray or plate to catch notes. Osu taiko, which is a game where you tap keys on a keyboard to a specific color on a drum, and last being Osu mania which is a game focused on hiting keys to a specific pattern, the game can be played with 1 to 10 keys.
